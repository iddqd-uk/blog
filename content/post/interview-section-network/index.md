## –ß—Ç–æ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –ø–æ—Å–ª–µ –≤–≤–æ–¥–∞ –∞–¥—Ä–µ—Å–∞ –≤ –±—Ä–∞—É–∑–µ—Ä?

–ê–¥—Ä–µ—Å (–ø–æ-–¥—Ä—É–≥–æ–º—É URL), –∫–æ—Ç–æ—Ä—ã–π –±—ã–ª –≤–≤–µ–¥—ë–Ω –≤ —Å—Ç—Ä–æ–∫—É –∞–¥—Ä–µ—Å–∞ —Å–æ—Å—Ç–æ–∏—Ç –∏–∑:

- –ü—Ä–æ—Ç–æ–∫–æ–ª–∞, –∫–æ—Ç–æ—Ä—ã–π –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –¥–æ—Å—Ç—É–ø–∞ –∫ —Ä–µ—Å—É—Ä—Å—É (`http`, `https`, `ftp` –∏ —Ç.–¥.)
  - –ï—Å–ª–∏ –ø—Ä–æ—Ç–æ–∫–æ–ª –≤–≤–µ–¥—ë–Ω –Ω–µ –±—ã–ª, —Ç–æ –±—Ä–∞—É–∑–µ—Ä —Å–º–æ—Ç—Ä–∏—Ç - –µ—Å—Ç—å –ª–∏ –æ–Ω —É –Ω–µ–≥–æ –≤ —Å–ø–∏—Å–∫–µ [HSTS][hsts] (HTTP Strict Transport Security, –º–µ—Ö–∞–Ω–∏–∑–º –ø—Ä–∏–Ω—É–¥–∏—Ç–µ–ª—å–Ω–æ –∞–∫—Ç–∏–≤–∏—Ä—É—é—â–∏–π –∑–∞—â–∏—â–µ–Ω–Ω–æ–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ —á–µ—Ä–µ–∑ –ø—Ä–æ—Ç–æ–∫–æ–ª `https`), –∏ –µ—Å–ª–∏ –¥–æ–º–µ–Ω –µ—Å—Ç—å —É –Ω–µ–≥–æ –≤ —Å–ø–∏—Å–∫–µ - –∑–∞–ø—Ä–æ—Å –±—É–¥–µ—Ç –æ—Ç–ø—Ä–∞–≤–ª–µ–Ω –∏—Å–ø–æ–ª—å–∑—É—è –ø—Ä–æ—Ç–æ–∫–æ–ª `https`, –∏–Ω–∞—á–µ - `http` (–≤–∫–ª—é—á–∞–µ—Ç—Å—è –æ–Ω —Å –ø–æ–º–æ—â—å—é HTTP –∑–∞–≥–æ–ª–æ–≤–∫–∞ –Ω–∞ —Å–µ—Ä–≤–µ—Ä–µ –¥–æ–º–µ–Ω–∞ `Strict-Transport-Security: max-age=31536000; includeSubDomains; preload;`)
- –•–æ—Å—Ç–∞ –∏–ª–∏ –¥–æ–º–µ–Ω–∞ (`example.com`, `linux.org`)
- –ù–æ–º–µ—Ä–∞ –ø–æ—Ä—Ç–∞ (–æ–ø—Ü–∏–æ–Ω–∞–ª—å–Ω–æ)
  - –î–ª—è –æ–±—â–µ–Ω–∏—è —Å –≤–µ–±-—Å–µ—Ä–≤–µ—Ä–æ–º –Ω–∞ –∑–∞–ø—Ä–∞—à–∏–≤–∞–µ–º–æ–º –¥–æ–º–µ–Ω–µ, –Ω–∞–º –Ω—É–∂–Ω–æ —É—Å—Ç–∞–Ω–æ–≤–∏—Ç—å TCP —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ —Å –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–º –ø–æ—Ä—Ç–æ–º. –í —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ –æ–Ω —É–∫–∞–∑–∞–Ω —è–≤–Ω–æ (`http://1.1.1.1:8080`) - —Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –æ–Ω, –∞ –∏–Ω–∞—á–µ - –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π –ø–æ—Ä—Ç –¥–ª—è –∑–∞–ø—Ä–æ—à–µ–Ω–Ω–æ–≥–æ –ø—Ä–æ—Ç–æ–∫–æ–ª–∞ (80 –¥–ª—è `http`, 443 –¥–ª—è `https`, –∏ —Ç–∞–∫ –¥–∞–ª–µ–µ)
- –ò –∑–∞–ø—Ä–æ—Å–∞ (query, —Ä–µ—Å—É—Ä—Å, URN) - —Å—Ç—Ä–æ–∫–∏ –∞–¥—Ä–µ—Å–∞ —Ä–µ—Å—É—Ä—Å–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä - –∞–¥—Ä–µ—Å —Å—Ç—Ä–∞–Ω–∏—Ü—ã)

### DNS

–ü–æ—Å–ª–µ —Ä–∞–∑–±–æ—Ä–∞ –∞–¥—Ä–µ—Å–∞ –º—ã –¥–æ–ª–∂–Ω—ã —É—Å—Ç–∞–Ω–æ–≤–∏—Ç—å —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ —Å —Å–µ—Ä–≤–µ—Ä–æ–º, –∏ —Å–¥–µ–ª–∞—Ç—å —ç—Ç–æ –ø–æ IP –∞–¥—Ä–µ—Å—É (–∞ –Ω–µ –¥–æ–º–µ–Ω–Ω–æ–º—É –∏–º–µ–Ω–∏, TCP/IP –∂–µ), –¥–ª—è —á–µ–≥–æ –Ω–∞–º –Ω—É–∂–Ω–æ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞—Ç—å –∏–º—è –¥–æ–º–µ–Ω–∞ –≤ IP –∞–¥—Ä–µ—Å _(–¥—Ä—É–≥–∏–º–∏ —Å–ª–æ–≤–∞–º–∏ - —Ä–∞–∑—Ä–µ–∑–æ–ª–≤–∏—Ç—å –∞–¥—Ä–µ—Å; –≤–æ–æ–±—â–µ, –¥–æ–º–µ–Ω—ã –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –≤ –ø–µ—Ä–≤—É—é –æ—á–µ—Ä–µ–¥—å –¥–ª—è —É–¥–æ–±—Å—Ç–≤–∞, –∏ –¥–ª—è –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ —Ä–∞–∑–º–µ—â–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å–∞–π—Ç–æ–≤ –Ω–∞ –æ–¥–Ω–æ–º –¥–æ–º–µ–Ω–µ)_. –î–ª—è —ç—Ç–æ–≥–æ:

- –°–ø–µ—Ä–≤–∞ –±—Ä–∞—É–∑–µ—Ä —Å–º–æ—Ç—Ä–∏—Ç –≤ —Å–≤–æ–π –∫—ç—à (–±—Ä–∞—É–∑–µ—Ä–∞)
- –ï—Å–ª–∏ —Ç–∞–º –Ω–µ –æ–±–Ω–∞—Ä—É–∂–µ–Ω–æ - —Ç–æ —Å–º–æ—Ç—Ä–∏—Ç –≤ —Ñ–∞–π–ª `hosts` (–≤ –∫–æ—Ç–æ—Ä–æ–º –º–æ–∂–Ω–æ –∑–∞—Ö–∞—Ä–¥–∫–æ–¥–∏—Ç—å –ª—é–±–æ–π IP –¥–ª—è –ª—é–±–æ–≥–æ –¥–æ–º–µ–Ω–∞)
- –ï—Å–ª–∏ –∏ —Ç–∞–º –Ω–µ—Ç, –Ω–æ —Å–º–æ—Ç—Ä–∏—Ç –≤ –∫—ç—à–µ –æ–ø–µ—Ä–∞—Ü–∏–æ–Ω–Ω–æ–π —Å–∏—Å—Ç–µ–º—ã (`systemd-resolve`)
- –ï—Å–ª–∏ –Ω–∏–≥–¥–µ –Ω–µ—Ç, —Ç–æ –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç—Å—è –∑–∞ –∞–¥—Ä–µ—Å–æ–º –≤ —Å–µ—Ç—å:
  - –û—Ç–ø—Ä–∞–≤–ª—è–µ—Ç—Å—è –∑–∞–ø—Ä–æ—Å –Ω–∞ —Å–µ—Ç–µ–≤–æ–π DNS —Å–µ—Ä–≤–µ—Ä (–∫–æ—Ç–æ—Ä—ã–π –±—ã–ª –ø–æ–ª—É—á–µ–Ω –ø–æ DHCP –∏–ª–∏ –ø—Ä–æ–ø–∏—Å–∞–Ω —Ä—É—á–∫–∞–º–∏)
    - –ï—Å–ª–∏ –∏ —É –Ω–µ–≥–æ –Ω–µ—Ç, —Ç–æ –æ–Ω (—Å–µ—Ä–≤–µ—Ä) –∏–¥—ë—Ç –≤ –∫–æ—Ä–Ω–µ–≤–æ–π —Å–µ—Ä–≤–µ—Ä, —Ç–æ—Ç - –≤ DNS —Å–µ—Ä–≤–µ—Ä –æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω—ã–π –∑–∞ –∑–æ–Ω—É (–Ω–∞–ø—Ä–∏–º–µ—Ä - `.com`) - –∏ —Ç–∞–∫ –¥–æ —Ç–µ—Ö –ø–æ—Ä, –ø–æ–∫–∞ —Ü–µ–ª–µ–≤–æ–π NS (–≥–¥–µ —Ö—Ä–∞–Ω—è—Ç—Å—è –∑–∞–ø–∏—Å–∏) –Ω–µ –±—É–¥–µ—Ç –Ω–∞–π–¥–µ–Ω
    - –ö–æ–≥–¥–∞ —Ü–µ–ª–µ–≤–æ–π NS —Å–µ—Ä–≤–µ—Ä –¥–æ—Å—Ç–∏–≥–Ω—É—Ç (—Ç–æ—Ç, –≥–¥–µ –∏ —Å–æ–¥–µ—Ä–∂–∏—Ç—Å—è –∏—Å–∫–æ–º–∞—è –∑–∞–ø–∏—Å—å –¥–ª—è –¥–æ–º–µ–Ω–∞; –æ–±—ã—á–Ω–æ —ç—Ç–æ `A` –∏–ª–∏ `CNAME`) - —Ç–æ IP –∞–¥—Ä–µ—Å —Å–µ—Ä–≤–µ—Ä–∞ (`A` –∑–∞–ø–∏—Å—å —Å–æ–¥–µ—Ä–∂–∏—Ç IP, –∞ –≤–æ—Ç —Å `CNAME` –∏—Å—Ç–æ—Ä–∏—è —Ä–µ–∑–æ–ª–≤–∏–Ω–≥–∞ –ø–æ–≤—Ç–æ—Ä—è–µ—Ç—Å—è –ø–æ –∫—Ä—É–≥—É, —Ç–∞–∫ –∫–∞–∫ –æ–Ω–∞ –ø—Ä–æ—Å—Ç–æ –ø–µ—Ä–µ–Ω–∞–ø—Ä–∞–≤–ª—è–µ—Ç –Ω–∞ –¥—Ä—É–≥–æ–µ –¥–æ–º–µ–Ω–Ω–æ–µ –∏–º—è) –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è –ø–æ —Ü–µ–ø–æ—á–∫–µ –æ–±—Ä–∞—Ç–Ω–æ (–¥–æ –∑–∞–ø—Ä–æ—Å–∏–≤—à–µ–π –µ—ë —Å—Ç–æ—Ä–æ–Ω—ã), –ø–æ–ø—É—Ç–Ω–æ –∫—ç—à–∏—Ä—É—è—Å—å –Ω–∞ –ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω—ã—Ö —Å–µ—Ä–≤–µ—Ä–∞—Ö (–µ—Å–ª–∏ —ç—Ç–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –∏ –≤–æ–∑–º–æ–∂–Ω–æ)

### –û—Ç–ø—Ä–∞–≤–∫–∞ –ø–∞–∫–µ—Ç–∞

–î–∞–ª–µ–µ, –∫–æ–≥–¥–∞ –Ω–∞–º –∏–∑–≤–µ—Å—Ç–µ–Ω IP –∞–¥—Ä–µ—Å —Å–µ—Ä–≤–µ—Ä–∞, –º—ã –º–æ–∂–µ–º —Å—Ñ–æ—Ä–º–∏—Ä–æ–≤–∞—Ç—å –∏ –æ—Ç–ø—Ä–∞–≤–∏—Ç—å –∑–∞–ø—Ä–æ—Å. –í—Å–ø–æ–º–∏–Ω–∞—è —É—Å—Ç—Ä–æ–π—Å—Ç–≤–æ —Å—Ç–µ–∫–∞ –ø—Ä–æ—Ç–æ–∫–æ–ª–æ–≤ TCP/IP –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç —Å–ª–µ–¥—É—é—â–µ–µ:

- –ù–∞ **–ø—Ä–∏–∫–ª–∞–¥–Ω–æ–º —É—Ä–æ–≤–Ω–µ** (application layer) –±—Ä–∞—É–∑–µ—Ä–æ–º —Ñ–æ—Ä–º–∏—Ä—É–µ—Ç—Å—è –∑–∞–ø—Ä–æ—Å –∫ —Å–µ—Ä–≤–µ—Ä—É (–±—É–¥—å —Ç–æ DNS, HTTP, HTTPS –∏ —Ç–æ–º—É –ø–æ–¥–æ–±–Ω–æ–µ)
- –î–∞–ª–µ–µ, –Ω–∞ **—Ç—Ä–∞–Ω—Å–ø–æ—Ä—Ç–Ω–æ–º —É—Ä–æ–≤–Ω–µ** (transport layer, `TCP` –∏–ª–∏ `UDP` –ø—Ä–æ—Ç–æ–∫–æ–ª) –≤ –∑–∞–≥–æ–ª–æ–≤–∫–∏ –ø–∞–∫–µ—Ç–∞ –¥–æ–±–∞–≤–ª—è–µ—Ç—Å—è –ø–æ—Ä—Ç, –ø–æ –∫–æ—Ç–æ—Ä–æ–º—É –Ω—É–∂–Ω–æ —Å—Ç—É—á–∞—Ç—å—Å—è
- –ù–∞ **—Å–µ—Ç–µ–≤–æ–º —É—Ä–æ–≤–Ω–µ** (network layer, `IP` –ø—Ä–æ—Ç–æ–∫–æ–ª) –¥–æ–±–∞–≤–ª—è–µ—Ç—Å—è IP –∞–¥—Ä–µ—Å –∫ –Ω–∞—à–µ–º—É –ø–∞–∫–µ—Ç—É
- –ù–∞ **–∫–∞–Ω–∞–ª—å–Ω–æ–º —É—Ä–æ–≤–Ω–µ** (data link layer) —Å –ø–æ–º–æ—â—å—é [ARP][arp] (Address Resolution Protocol, –ø—Ä–æ—Ç–æ–∫–æ–ª –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω–Ω—ã–π –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è MAC-–∞–¥—Ä–µ—Å–∞ –¥—Ä—É–≥–æ–≥–æ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞ –ø–æ –∏–∑–≤–µ—Å—Ç–Ω–æ–º—É IP-–∞–¥—Ä–µ—Å—É) –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è "–µ—Å—Ç—å –ª–∏ —Ç–∞–∫–æ–π –∞–¥—Ä–µ—Å –≤ —Å–µ—Ç–∏?"
  - –ï—Å–ª–∏ –∞–¥—Ä–µ—Å –µ—Å—Ç—å –≤ —Å–µ—Ç–∏ - —Ç–æ –∫ –Ω–µ–º—É –ø–µ—Ä–µ–¥–∞—ë—Ç—Å—è –ø–∞–∫–µ—Ç
  - –í –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ –ø–∞–∫–µ—Ç –ø–µ—Ä–µ–¥–∞—ë—Ç—Å—è –Ω–∞ —à–ª—é–∑, –∫–æ—Ç–æ—Ä—ã–π, –ø–æ–¥–Ω–∏–º–∞—è—Å—å –ø–æ —É—Ä–æ–≤–Ω—è–º –≤—ã—à–µ (—Å–º–æ—Ç—Ä—è –Ω–∞ —Ç–æ, –∫–∞–∫–æ–π IP –∞–¥—Ä–µ—Å –∏ —Å–º–æ—Ç—Ä—è —Å–≤–æ—é —Ç–∞–±–ª–∏—Ü—É –º–∞—Ä—à—Ä—É—Ç–∏–∑–∞—Ü–∏–∏) - –Ω–∞–ø—Ä–∞–≤–ª—è–µ—Ç –ø–∞–∫–µ—Ç –≤ –Ω—É–∂–Ω–æ–º –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏–∏ (–ø–æ–∫–∞ –Ω–µ –∑–∞–∫–æ–Ω—á–∏—Ç—Å—è TTL –ø—Ä–æ—Ç–æ–∫–æ–ª–∞ IP, –ª–∏–±–æ –ø–∞–∫–µ—Ç –Ω–µ –¥–æ—Å—Ç–∏–≥–Ω–µ—Ç –ø—É–Ω–∫—Ç–∞ —Å–≤–æ–µ–≥–æ –Ω–∞–∑–Ω–∞—á–µ–Ω–∏—è)

### TLS Handshake

–ï—Å–ª–∏ –∑–∞–ø—Ä–æ—Å –±—ã–ª –æ—Ç–ø—Ä–∞–≤–ª–µ–Ω –ø–æ –ø—Ä–æ—Ç–æ–∫–æ–ª—É `https`, —Ç–æ –Ω–∞—Å—Ç—É–ø–∞–µ—Ç –ø–æ—Ä–∞ —É—Å—Ç–∞–Ω–æ–≤–∏—Ç—å –∑–∞—â–∏—â–µ–Ω–Ω–æ–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ (TLS Handshake), –¥–ª—è —ç—Ç–æ–≥–æ:

- –ë—Ä–∞—É–∑–µ—Ä –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç `Client hello` —Å–µ—Ä–≤–µ—Ä—É —Å–æ —Å–≤–æ–µ–π –≤–µ—Ä—Å–∏–µ–π –ø—Ä–æ—Ç–æ–∫–æ–ª–∞ [TLS][tls] (–Ω–∞–ø—Ä–∏–º–µ—Ä - `TLSv1.3`)
- –°–µ—Ä–≤–µ—Ä –æ—Ç–≤–µ—á–∞–µ—Ç –∫–ª–∏–µ–Ω—Ç—É —Å–æ–æ–±—â–µ–Ω–∏–µ–º `Server hello`, —Å–æ–¥–µ—Ä–∂–∞—â–∏–º –≤–µ—Ä—Å–∏—é TLS, –≤—ã–±—Ä–∞–Ω–Ω—ã–º –º–µ—Ç–æ–¥–æ–º —à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—è, –º–µ—Ç–æ–¥–æ–º –∫–æ–º–ø—Ä–µ—Å—Å–∏–∏ –∏ –ø—É–±–ª–∏—á–Ω—ã–º —Å–µ—Ä—Ç–∏—Ñ–∏–∫–∞—Ç–æ–º —Å–µ—Ä–≤–∏—Å–∞ (–ø–æ–¥–ø–∏—Å–∞–Ω–Ω—ã–π —Ü–µ–Ω—Ç—Ä–æ–º —Å–µ—Ä—Ç–∏—Ñ–∏–∫–∞—Ü–∏–∏; —Å–µ—Ä—Ç–∏—Ñ–∏–∫–∞—Ç —Å–æ–¥–µ—Ä–∂–∏—Ç –ø—É–±–ª–∏—á–Ω—ã–π –∫–ª—é—á, –∫–æ—Ç–æ—Ä—ã–π –±—É–¥–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –∫–ª–∏–µ–Ω—Ç–æ–º –¥–ª—è —à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—è)
